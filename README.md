# Peru Driving Test - Practice App

A vibe-coded Progressive Web App for practicing Peru driver's license A-1 test questions.

## Features

✅ **Smart Question Selection** - Questions you struggle with appear more frequently  
✅ **Multiple Learning Modes** - Random, Sequential, and Simulated Test modes  
✅ **Simulated Test Timer** - 40 minutes countdown with visual warnings  
✅ **Skip Questions** - Skip questions you don't know and come back to them later  
✅ **Progress Tracking** - Track your performance and accuracy  
✅ **Dark/Light Theme** - Toggle between themes for comfortable studying  
✅ **Offline Support** - Works without internet after first load  
✅ **Installable** - Install as an app on mobile and desktop
